# Expense Tracker Project

## Description 

A simple Expense Tracker project built using Python.   THis project allows users to add expenses, view expense historyand calculate the total expense. It also includes input validation to improve data accuracy and user experience.

## Feature

* Add new expenses
* View expense history
* Calculate total expenses
* Date validation (DD-MM-YYYY format)
* Item name validation
* Amount validation
* Handles invalid input using try-except
* Numbered and formatted expense history

## Technologies Used

* Python 3
* datetime module

## How to Run

1. Download or clone this repository.
2. Open the project in VS Code.
3. Run the Python file.
4. Choose an option from the menu.
5. Add, view, or calculate your expenses.
